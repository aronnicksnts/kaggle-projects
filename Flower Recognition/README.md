# Abstract - Flower Recognition
With a dataset of 4242 images of flowers, this project utilizes the knn algorithm in order to predict if the flower is either a **daisy**, **dandelion**, **rose**, **sunflower**, or a **tulip**. With the knn algorithm the model was approximately 43% accurate.
## Dataset Link
https://www.kaggle.com/datasets/alxmamaev/flowers-recognition
